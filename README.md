## Murder at the Occult Club

A browser-based murder mystery puzzle game built on Next.js and deployed with Vercel.

Play the game here! https://murder-at-the-occult-club.vercel.app/ The game takes about 1 hour to complete (shorter if you're really smart).

### Current Features

- A full-fledged storyline, characters, and motivations
- 8 evidence types with about 30 total pieces of evidence to investigate
- Fill-in-the-blank-style puzzles that unlock more evidence as you go on
- A save system that automatically keeps track of your progress

### Planned Features

- A feedback form to submit feedback/bugs
- Sound effects and images to enhance immersion
